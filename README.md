# 🍎 AIET CSE iOS Club Website

A futuristic and premium Apple-inspired website developed for the **AIET CSE iOS Club** using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Apple-inspired modern UI
* Responsive design
* Smooth scrolling animations
* Interactive project showcase
* Neon glow effects
* Glassmorphism design
* Dynamic navigation bar
* Event timeline section
* Team showcase
* Social media integration
* Mobile-friendly layout

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎨 Design Style

The website is inspired by:

* Apple Event Pages
* WWDC UI Design
* Futuristic Cyberpunk Interfaces
* Modern Startup Landing Pages
